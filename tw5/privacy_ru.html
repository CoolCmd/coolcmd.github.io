Политика конфиденциальности описывает сбор и использование данных расширением браузера Alternate Player for Twitch.tv. Разработчик расширения — Alexander Choporov (CoolCmd).

Сбор данных

На сбор данных не влияет режим икогнито браузера Google Chrome и приватный просмотр браузера Mozilla Firefox. Расширение не работает в режиме InPrivate браузера Microsoft Edge.

Расширение хранит на компьютере пользователя следующие данные:

  • Настройки проигрывателя, которые пользователь может менять в морде проигрывателя.
  • Данные, необходимые для правильной работы расширения, например номер версии расширения.
  • Случайное число, которое используется в качестве уникального идентификатора пользователя. Идентификатор необходим, чтобы отделить ошибки, возникающие на конкретной конфигурации компьютера.

Во время своей работы проигрыватель собирает отладочные данные, в которых подробно описана вся работа, проделанная проигрывателем в течении последних минут. Отладочные данные содержат только анонимную информацию (насколько это возможно). После закрытия страницы с проигрывателем отладочные данные уничтожаются.

Если во время работы расширения возникла ошибка или пользователь захотел отправить отзыв о работе расширения, то на интернет-сервер, арендуемый разработчиком расширения, отсылаются следующие данные:

  • Произвольный текст, введенный пользователем.
  • Адрес электронной почты пользователя (вводить не обязательно).
  • Данные расширения, хранящиеся на компьютере пользователя (описаны выше по тексту).
  • Отладочные данные (описаны выше по тексту).
  • При необходимости, несколько последних секунд просмотренного видео.

Пользователь может просмотреть содержимое отправляемых данных и отказаться от их отправки.

Инернет-сервер собирает следующие данные:

  • IP-адрес компьютера, пославшего данные.
  • Заголовки запроса, такие как UserAgent и AcceptLanguage.

Использование собранных данных

Собранные данные хранятся на компьютере разработчика расширения и используются исключительно для улучшения работы расширения. Третьим лицам данные не передаются. Адреса электронной почты не используются для рассылки рекламной и иной информации, не связанной с работой расширения. По окончанию использования, собранные данные уничтожаются.

Данные, собранные интернет-сервером (описаны выше по тексту), хранятся на нём в виде логов и используются для улучшения работы интернет-сервера. Третьим лицам данные не передаются.
