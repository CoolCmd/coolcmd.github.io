Privacy Policy describes the collection and use of data in the browser extension Alternate Player for Twitch.tv. Developer of the extension — Alexander Choporov (CoolCmd).

Collecting data

The data collection is not affected by the incognito mode of the Google Chrome browser and private browsing of the Mozilla Firefox browser. The extension does not work in InPrivate mode of the Microsoft Edge browser.

Extension stores the following data on the user's computer:

  • Player settings that the user can change in the interface of the player.
  • The data required for proper operation of extensions, such as the extension version number.
  • A random number that is used as a unique user identifier. The identifier is required to detect errors occurring in a particular computer configuration.

During its work, the player collects debug data, detailing all the work done by the player during the last few minutes. Debug data contains only anonymous information (when possible). After closing the page with the player, the debug data will be destroyed.

If an error occurred in the extension or a user wants to send feedback, then following data will be sent to the web server:

  • Arbitrary text entered by the user.
  • E-mail address of the user (optional).
  • The extension's data which is stored on the user's computer (described above).
  • Debug data (described above).
  • If necessary, the last few seconds of the video viewed.

The user can view the contents of the data sent and refuse to send them.

Web server collects the following data:

  • The IP address of the computer that sent the data.
  • Request headers, such as UserAgent and AcceptLanguage.

Using collected data

The collected data is stored on the computer of the developer of the extension and is used solely to improve the work of extension. The data will not be sent to a third party. E-mail addresses are not used for sending advertising and other information which isn't related to extension work. After use, the collected data will be destroyed.

Data collected by the web server (described above) is stored in the server logs and used to improve web server performance. The data will not be sent to a third party.
